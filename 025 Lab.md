
# install VM instance with virtual box
[How to install Ubuntu 22.10 LTS in VirtualBox 2026](https://www.youtube.com/watch?v%3DhYaCCpvjsEY)

- install virtual box
- download tiny coreLinux: https://sourceforge.net/projects/tinycoremeta4s/
- build the VM 
- use : tiny{yourname} as the name for the VM
- Run VM and send and screenshot with the terminal open on the screen and showing the VM name
